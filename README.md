# stm32-cw-keyer
Simple embeddable CW Keyer (Elekey-B type).

Pinout:

- PA0: speed control (about 10-35 WPM)
- PA1: dit / straight key input
- PA2: dah input
- PA3: output
- PA9: pull to GND for straight key mode
